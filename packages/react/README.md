# @agentclutch/react

React-compatible components for AgentClutch Action Cards and takeover UX.

This package is part of [AgentClutch](https://github.com/MelaBuilt-AI/agentclutch): the local-first Action Card and takeover UX layer for consequential AI agent actions.

## Status

`0.7.3-alpha.0` is the first published npm alpha version. Prefer installing with the explicit `@alpha` tag until AgentClutch is stable.

## Install

```bash
pnpm add @agentclutch/react@alpha
```

## Usage

```ts
import { ActionCard } from "@agentclutch/react";
```

## Package role

Reusable Action Card and Run Story UI components.

See the root AgentClutch README and `docs/quickstart.md` for full setup, demos, and caveats.
