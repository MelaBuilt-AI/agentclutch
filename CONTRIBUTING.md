# Contributing

AgentClutch is built as a TypeScript pnpm monorepo.

Before changing code:

1. Read the relevant section of `AgentClutch.md`.
2. Keep changes scoped to the current task.
3. Add tests where product behavior is added.
4. Run `pnpm build`, `pnpm test`, and `pnpm typecheck`.

Use Node.js 22 or newer.

## Filing issues

Use the GitHub issue templates for bug reports, feature requests, and docs improvements. Include the package or area, AgentClutch version, install method, OS, Node version, pnpm version, and small reproduction steps when relevant.

Please remove secrets and private data from Action Cards, recorder JSONL, Run Stories, screenshots, and logs before posting them publicly.
